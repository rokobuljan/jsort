JSort, your JavaScript library for sorting and reordering items in a list or grid.

GitHub rokobuljan/jsort
NPM @rbuljan/jsort

Grid

Sortable CSS grid.
Drop an item on another item, the order of all other items changes with animation once the drop finishes for improved cognitive load.

Drop onto parent is also supported.

Need a grab handler? Add a child element with the class jsort-handler to the item. You can change that class name from the instance Options

Nested

Nested sortables are also supported.
Add a data-jsort-group="name" to your nested elements and assign a JSort instance to every one of them. PS, you can also drag an item to the above Groups ;)

    • C 1 a
    • C 1 a 1
      • D.a 1
      • D.a 2

Swap

Swap items between two lists.
Add data-jsort-swap="true" attribute to your grouped lists.
